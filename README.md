# Hotel Management System

A full-stack hotel management system with React frontend and Node.js backend.

## Project Structure

```
hotelmgmt/
├── client/          # React frontend
│   ├── src/
│   ├── public/
│   └── package.json
│
└── server/          # Node.js backend
    ├── src/
    ├── config/
    └── package.json
```

## Setup Instructions

### Frontend (Client)
```bash
cd client
npm install
npm run dev
```

### Backend (Server)
```bash
cd server
npm install
npm run dev
```

## Features
- User authentication with Clerk
- Room management
- Booking system
- Hotel owner dashboard
- Responsive design
